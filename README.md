# AI Fresher Hiring Selection Prediction

## Project Status

🚧 Work in Progress

This project is being developed, more visualizations and analysis will be added in future updates.

## Overview

The goal of this project is to build a Logistic Regression model that predicts whether an engineering fresher is likely to be selected based on academic performance, technical skills, projects, certifications, internships, and assessments.

## Dataset Features

* Age
* CGPA
* Aptitude Score
* Programming Score
* SQL Score
* Communication Score
* Projects
* Certifications
* Internship Months
* Hackathon Participation
* Attendance
* Selected (Target Variable)

## Tasks Completed

* Data Loading
* Dataset Inspection
* Data Type Analysis
* Missing Value Analysis
* Summary Statistics
* Percentage of Selected Candidates
* Train-Test Split
* Logistic Regression Model
* Model Evaluation using Accuracy Score
